# PWA Icons Required

To complete the PWA setup, you need to create the following icon files in this directory:

## Required Icon Sizes:
- `icon-16x16.png` (16x16 pixels) - Browser favicon
- `icon-32x32.png` (32x32 pixels) - Browser favicon
- `icon-72x72.png` (72x72 pixels) - PWA icon
- `icon-96x96.png` (96x96 pixels) - PWA icon
- `icon-128x128.png` (128x128 pixels) - PWA icon
- `icon-144x144.png` (144x144 pixels) - PWA icon
- `icon-152x152.png` (152x152 pixels) - PWA icon
- `icon-192x192.png` (192x192 pixels) - PWA icon (minimum required)
- `icon-384x384.png` (384x384 pixels) - PWA icon
- `icon-512x512.png` (512x512 pixels) - PWA icon (maximum recommended)
- `maskable-icon-512x512.png` (512x512 pixels) - Maskable icon for Android

## Icon Design Guidelines:
1. **Theme**: Agricultural/IoT monitoring theme with green and blue colors
2. **Elements**: Consider including:
   - Plant/leaf symbol
   - Sensor/monitoring device
   - Water droplet or irrigation symbol
   - Circuit/tech elements
3. **Colors**: Primary blue (#1976d2), green (#4caf50), white background
4. **Format**: PNG with transparent background where appropriate
5. **Maskable Icon**: Should have important elements in the center "safe zone"

## Tools for Creating Icons:
- **Online**: Canva, Figma, Adobe Express
- **Software**: Adobe Illustrator, GIMP, Sketch
- **PWA Icon Generators**: 
  - https://www.pwabuilder.com/imageGenerator
  - https://appsco.pe/developer/splash-screens
  - https://maskable.app/ (for maskable icons)

## Quick Setup:
1. Create a base 512x512 icon design
2. Use an icon generator tool to create all required sizes
3. Replace this README with the actual icon files
4. Test the PWA installation on mobile devices

The PWA will work without these icons, but they significantly improve the user experience and make the app look professional when installed on devices.
